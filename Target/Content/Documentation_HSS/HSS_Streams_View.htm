HyperStream - Streams View

The Stream View shows all the streams stored on the . It can be viewed only by an administrator. Each Stream represents a backup.

Connect to the with administration rights.

Select HyperStream > Accessible Streams.

(Optional) Depending on the average size of your stream, you may want to change the size unit used. Click the arrow in the upper right corner of the Streams view to select a unit from Bytes to Tera Bytes.

(Optional) To reload the Streams view, click the Refresh button.

For each stream, the following columns are displayed:

Stream UUID Identifier of the stream.

Backup Date Date of the backup contained in the stream.

Agent Name of the client backed up.

Source Source of the stream.

Status Status of the stream.

Pool Media pool including the stream.

Size Volume of the stream (volume of data read on the client during backup).

Deduplication Ratio Deduplication ratio per stream.
