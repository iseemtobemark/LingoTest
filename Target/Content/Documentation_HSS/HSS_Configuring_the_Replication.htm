Configure the Replication

Ensure that you installed a server on all the machines involved.

To configure a simple or a hierarchical replication, you must create a replication destination on every source server involved.

To configure bidirectional replication, which is a two way replication, you must create a replication destination on both the source server, and the target server.

The replication of the source server starts as soon as the replication destination is created. Ensure that the time to launch the replication is appropriate prior to creating the replication destination.

To create a replication destination for simple or hierarchical replication

In the of the server you want to replicate, select Server Configuration   Replication.

Click the New Replication Destination icon located at the top of the Replication view.

Enter the network name of the replication target server and its port number.

Click Apply and Close to add the replication destination.

To configure a bidirectional replication between Server A and Server B

In the of Server A, select Server Configuration   Replication.

Click the New Replication Destination icon located at the top of the Replication view.

Enter Server B name and port number and click Apply and Close.

In the of Server B, select Server Configuration   Replication.

Click the New Replication Destination icon located at the top of the Replication view.

Enter Server A name and port number and click Apply and Close.
