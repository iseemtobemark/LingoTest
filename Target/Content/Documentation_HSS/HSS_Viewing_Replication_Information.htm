Viewing Replication Information

You can display replication information in the Agent view of the and in command mode.

To display replication information in the

In the , select    Agents.

Click Columns to Display. Check the replication columns you want to display.

This table describes the replication information available for each agent:

Column

Description

Replication Source Name

Name of the server providing data for the specific agent. If the source is the agent (or if no replication is defined), this field is empty.

Replication Destination Name

Name of the server to which the current server is replicated. If you are viewing the from the replication destination server (or if no replication is defined), this field is empty.

Last Synchronization Point Replicated

Time of the last synchronization, i.e., last protection time of the replicated data.

Replication Lag

Displays information regarding the replication of this agent. Synchronized means the agent has been correctly replicated to the target replication server.

To display replication information in command line

On the machine hosting the :

Windows.

Select Start   Programs   Atempo-Deduplication Engine   Administration.

Linux.

Launch the ./ade_admin command.

Provide a password. Two default users exist in the :

admin, password admin (default user with administrative rights)

tina, password tina (standard user)

If you must launch the Command Line Interface with a different user, then run the ./ade_admin -u username command.

You are prompted to enter the password for this user.
