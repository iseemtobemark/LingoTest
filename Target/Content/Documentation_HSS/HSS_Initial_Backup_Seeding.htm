Initial Backup Seeding

This topic describes how to perform the initial backup of an agent connected through a slow link to an HyperStream repository (i.e., seeding).

The initial backup of a remote agent consists of:

Backing up the remote agent to a temporary directly connected to the same LAN.

Moving the to the central site, and replicating it to the main .

Re-runing a full backup from the remote agents so that appropriate metadata is recorded in the catalog.

To perform the initial backup of a remote agent

Install an on a mobile machine (named B).

Configure the replication on B to replicate any new stream to the main (named A) in the central site. See Simple and Hierarchical Replication for details.

Stop the replication service on B (to prevent any new stream on B from being copied onto A while server B is away).

Create a new HyperStream drive on the server and set it to connect to the B.

Create a new media pool and link it to the newly created HyperStream drive connected to the B.

Create a new strategy for every remote host agent to be backed up, using the newly created HyperStream drive.

Go to the remote site and connect the mobile B to the local LAN. At this point, the host(s) to be backed up and B are on the same LAN.

Do the first full backup of the host(s) located on the current LAN using the strategy you created and observe the new streams getting created on  B.

Repeat step 7 and step 8 for every remote site.

Bring the mobile  B back onto the main LAN where the  A is connected.

Start the replication service on the server B so that all the streams previously created on it are copied onto the server A.

Create a new media pool and associate it with the HyperStream drive(s) connected to the  A.

Modify the strategy created earlier on all the remote host agents in order to have it use the media pool created in step 12.

Perform a new full backup for every remote agent.

Remove the  B from the loop and delete the HyperStream drive associated with the  B as well as the media pool associated with this drive.
