Synchronization Frequency

By default, synchronization is performed every hour. To change the frequency, change the value of the tunable: hss_synchronization_interval.

To change the synchronization frequency

In the , expand your catalog and select Configuration  Tunables.

Click the New Tunable button. In the Tunable Creation Wizard, set the hss_synchronization_interval tunable.

Possible values for the tunable are:

hourly (default)

daily

disable

If you disable automatic synchronization, you can still perform manual synchronizations in the by running the tina_odbgc -hss_sync -catalog <catalog_name> command.
