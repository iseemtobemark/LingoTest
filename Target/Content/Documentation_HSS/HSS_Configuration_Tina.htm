Enter the size in MB of the medias used to store the backed up data. The size value you enter should be the same as the physical tape size that will be used for duplication to tape if you intend to duplicate the disk backup to tape.

To avoid segmenting backups on 2 cartridges, the cartridge size may be increased by . For instance, if you select a size of 100 GB, with only 10 GB left on a cartridge, and that a 20 GB backup is launched, the size of the cartridge will be increased to 110 GB to fit the entire backup job on the same cartridge.

Compression Algorithm

Name of the algorithm used to compress the data and metadata of the HyperStream clients. Possible values are:

Zlib

Lzma

Lzf

In general, Brotli is recommended in a LAN environment whereas Lzma is recommended in WAN environments. Lzf is a rapid algorithm with average compression.

If you want to apply compression on a host but not on another host, you must use separate Hyperstream drives and separate pools for these hosts.

If you want to apply compression on a host but not on another host, you must use separate Hyperstream drives and separate pools for these hosts.

If you are deduplicating a backup selection on which the compression option is checked, automatically disables the compression and uses the Hyperstream compression instead.

Number of drives to create

Specify the number of drives that must be created.

You should create as many drives as you want jobs to be deduplicated in parallel. For instance, if you have 10 drives, the HyperStream Server is able to deduplicate 10 backups at the same time.

Create a Dedicated Media Pool

To create a dedicated media pool for HyperStream backups, you must:

Use this pool only for HyperStream backups.

Avoid mixing regular backups with deduplicated backups.

Use the pool by a single HyperStream user with a single .

To create a media pool for HyperStream backups

In the , select Backup Destinations > Media Pools.

In the Media Pool view, click the New Pool button (+).

Provide the following information:

Pool Name Enter the name of your choice for the pool.

Pool Label Specify the label that appears in the name of all the cartridges belonging to the pool. It is usually a short version of the pool name.

Comment (optional) Enter a comment to describe the media pool. The comment is recorded in the catalog.

Retention Period and Duration Once a cartridge is filled, define the period of time that waits to recycle the cartridge:

Infinite Select this option if you do not want to recycle cartridges. In that case, does not reuse a cartridge once it is filled.

Limited Select this option if you want to recycle cartridges after a set period of time. Enter a numerical value and select a unit between Years, Months, Weeks, and Days.

Associated Drives Drives associated to the same pool must have the same parameters, i.e., same HyperStream Server, same port number, same user and same block size. A verification is performed and all incompatible drives are set to maintenance mode.

Click Choose Objects to open the Drive Name List.

Select all the HyperStream drives to associate with the pool.

Click Apply and Close.

Click Finish and Close to create the pool admin and exit the Wizard. See the for details.

Create a Strategy

In the , expand your catalog and select Data Management > Strategies.

Click the New Strategy button (+).

Select New Standard Strategy. In the creation wizard, follow on-screen instruction. See the for details on creating a strategy.

Select media pools that are dedicated to HyperStream.

Click Finish to create the Strategy.

In the Strategies view, double click the strategy you just created to see the details and make sure that:

In the Option area, Use cache on Server is not selected.

In the Advanced area, the writing format is SIDF. This is the only writing format supported by the HyperStream drives.

Set Tunables

In the , expand your catalog and select Configuration > Tunables.

Click the New Tunable button.

In the Tunable Creation Wizard, set the dedup_sync_for_canceled_jobs= advanced tunable to yes. See the for details.

Change Drives Port Numbers

You may need to change HyperStream drives port numbers if you change the port number on your . Drives port numbers must always match the server port number.

Select the drive in the Storage > Drive > Drive List.

Click the Drive Maintenance button.

Double click the drive and select Properties.

In the General area, modify the Port Number and click OK. Repeat for all the drives you want to modify.

Enable the drives.
