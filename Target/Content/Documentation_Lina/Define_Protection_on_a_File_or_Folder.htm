Define Protection on a File or Folder

The backup settings include the possibility to define protection on certain files or folders of your choice (see Figure).

The block on the right contains the agent backup settings

To define protection on a file or folder

Right-click the agent icon in the task bar.

Select .

The list of protections is displayed.

In My backup settings, click the + icon to select a file or a folder. The explorer opens.

Select the item(s). The folder or file is added to the list of items included in / excluded from the backup.

To delete an item, select it from the list and click the recycle bin to confirm removal. To update the list of items, click the Refresh button beside the recycle bin.
