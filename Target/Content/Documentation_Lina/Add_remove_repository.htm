Add or Remove a Block File Repository

To add or remove a repository, you have to:

Open the .

Select Server > Preferences.

Click on the Tunables tab.

Double-click the variable repository.path.

The variable description window appears on the right.

To add a repository, enter a semicolon followed by the new repository path after the existing repository path.

If you define several repositories, blocks are copied successively into one or the other repository. The server writes the first block file into the first repository, then the next block file into the next repository, etc.

To remove a repository, delete the path of the repository you want to remove.

Removed repositories that contained backups, are kept in the repository unless you move the block files using the ade_server command.

Close the window or click outside to validate the new value.
