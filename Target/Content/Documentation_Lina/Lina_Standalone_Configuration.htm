A standalone configuration is composed of a standalone server, and several tenants and agents.

The Server:

Stores protected data from the agents according to protection policies defined by the administrator and the user.

Relies on the (ADE) to store all the unique data blocks sent by the agents. The ADE database keeps track of all the data blocks stored, and the repository where data is actually stored.

The Agent:

Detects new and modified data.

Applies the protection policy set on the server side by the administrator and on the agent side by the user.

Sends unique blocks of protected data to the Server, according to the protection policies.

Provides restore capability, through a or Windows Explorer.
