Access the Monitoring Interface

The dashboard (see Figure) enables you to view information about the current configuration of the , and to view the protection activity in real time.

Right-click on the icon and select .

Dashboard

Each dashboard card in the interface presents information about the protection of the agent (see Table):

Agent dashboard parameters

Dashboard card

Description

Protection

Indicates the protection status of the . These are the possible statuses:

Protected The agent is fully protected. There is no new modifications to send to the server.

Processing The agent is sending objects to the server.

Analyzing The agent is detecting changes.

Modifications detected The agent has detected new or modified data to back up.

Protection paused You have suspended the protection.

Disconnected The agent cannot reach the server.

Resynchronizing The agent database is resynchronized from the server.

Network paused The network connection is blocked due to a network option set. See Configuring Network and Bandwidth for details.

Missing CDP module The agent cannot access the CDP module which is supposed to be present on the platform.

Scan Mode The agent is running on a platform which does not support CDP.

Indicates the status of the server. These are the possible statuses:

Active The server is protecting the agent.

Quota limit reached The quota set for this agent has been reached. The server will not protect new data.

Agent unauthorized The agent is not declared on the server.

Maintenance The server is busy and is unable to back up the agent data.

Date and time when agent files were last sent to the server.

Number of protected files.

Processing

Displays in a graph the backed up data in KB/s or MB/s.

Network activity

Displays in a graph the outgoing network traffic in KB/s.

Backup

Displays the backup activity on the . You can click on the dashboard card to view the list of backed up files.

Current Item Path and name of the object which is being backed up.

Last Item Path and name of the last backed up objects.

General information

Displays the following information:

Agent name Name of the host.

Server name Name of the server host.

Recovery Point Objective (RPO) Defines the maximum time lapse between two protection sessions. The RPO indicates the maximum acceptable data loss if the data is restored between two sessions.

Compression Indicates the compression algorithm as defined in the Settings. See Configuring Network and Bandwidth for more information.

Protection for BMR Indicates a module to recover a complete Windows agent.

Network information

Displays the following information:

Last connection Indicates the date and time when the agent last connected to the server.

Interface type Indicates the type of network interface used (e.g., Ethernet, Wi-Fi).

Gateway Indicates the gateway through which the connects to the server.

Encryption Indicates if data is encrypted or not.

Bandwidth limit Indicates the amount of bandwidth used for data transfer, as defined in the Settings. See Configuring Network and Bandwidth for more information.
