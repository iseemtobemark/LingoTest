Restoring the Server

To restore the server

Stop the before attempting to restore the database. The restore is canceled automatically if the server is running.

Windows.

Through the Start  Administrative Tools  Services menu or with this command

net stop ADE.default

Unix.

With this command

/etc/init.d/ade.default stop

Open the . Even if there are problems with your installation, the Synchronizer is still shown. This diagram illustrates the Synchronizer object in the file tree:

 

Right-click the Synchronizer and select the backup to be restored. Select the entry, unselect the Synchronizer object, and select Restore  Run.
