Restore a File from an Explorer

You can restore a previous version of a folder if :

The file still exists but you want to recover it as it was at an earlier date.

You know where the file is located.

You want to restore the file to a new location.

An earlier version already exists in the destination folder, and you want to rename the restored version.

To restore a previous version of a file from an Explorer

Right-click the file in an Explorer and select Backup > Show file.

In the list of available file versions, select the version to restore by selecting the desired date.

Click Restore.

Choose the restore location. By default, the file is restored to its original location, but you can switch the toggle on to restore the file to another location.

In the field If the file already exists, select Rename.

Click Restore. To cancel the restore process, click Cancel.

The already existing file is displayed with the extension .bak.
