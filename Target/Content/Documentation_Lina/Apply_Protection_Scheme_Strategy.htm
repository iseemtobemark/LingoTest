Apply a Protection Scheme and Strategy to an Agent

Open the .

Select Agents to display the list of agents.

In the list of agents, select the agent to modify.

Click the pencil button.

Use the Protection and Strategy drop-downlists to modify the agent settings.

Click Save.

The data to protect on your is now protected by the server, using the protections and strategies you selected.

You can also create a new protection or strategy and apply it to the agent using the + button.
