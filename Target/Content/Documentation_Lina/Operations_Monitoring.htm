Operations Monitoring

The Operations Monitoring displays information on connections and running operations (see Table).

Monitoring parameters
Parameters Description
Operation The type of operation
Agent The agent on which the operation is performed
Idle Time Period of time during which there is no activity
Workstation Name Name of the connected machine
Client/Agent Name Host name of the connected machine
IP Address IP address of the connected machine
Task Running task in progress
Requests Requests in progress on the machine
Connection date Date of the connection to the machine
