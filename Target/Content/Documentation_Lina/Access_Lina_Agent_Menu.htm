Access the Agent Menu

The menu is your access point to manage data protection. You can find it:

in the system tray for Windows.

in the menu bar for Mac.

To open the menu

Right-click on the   icon.

It enables to:

Restore data from the agent interface.

Access the dashboard.

Manage the settings.

View the help section.

Perform actions to pause or accelerate data protection.

When holding the pointer over the agent icon, you can check the most recent protection and see the volumes and number of files protected.
