Configure the Server Port for from Version 4.0.x Server Port

This topic only applies to versions of up to 4.0.x.

See Separate Ports Configuration for Administration and Data for starting with version 5.

To redefine port numbers:

In the , select Server > Preferences and click .

Select the option Show advanced parameters.

Double click on the variable aln.http_port to modify the HTTP port or aln.https_port to modify the secured HTTPS port. The field can be edited.

Enter the new port value in the edit window.

Click outside of the field to validate the new value.

Restart the service.
