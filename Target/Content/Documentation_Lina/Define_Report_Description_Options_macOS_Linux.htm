Define the Report Description Options on macOS or Linux

Option 1: Generate Configuration Only

Option 2: Generate a Compressed File

This option, set by default to Yes, causes the Environment Report to be compressed into a gzip file on generation. This action reduces the report size significantly and makes it easier to send or upload.
