Log into Web Restore as End User

To perform a as an end user, you need to log into the application.

In your browser, enter the following URL:

https://<your_server_name>:8181/webrestore.

 

http://<your_server_name>:8181/webrestore if you are on a secure local network.

If the dual port is configured, use the data port.

Type the e-mail address of the agent from which you want to restore data and click Next.

Select Request a security code to receive a session password by e-mail to the address provided in the previous step.

Enter the security code.

Select Login to access the application.
