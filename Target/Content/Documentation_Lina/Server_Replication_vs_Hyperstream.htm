Server Replication vs AgentHyperstream Replication

If you enable server replication, existing AgentHyperstream specific replication processes are disabled, even though their configuration is kept. Server replication configuration takes precedence over the other replication processes available, and its parameters override existing AgentHyperstream specific replication parameters:

The target server is the one defined in the server replication.

The connection is never encrypted.

The port is the one set by the HN_TARGET_HTTP_PORT tunable. See Replication Prerequisites for details on this tunable.

No proxy is used.
