Create Agents automatically

The Agent auto-creation option allows you to:

Automatically add all the agents associated with the server in the as soon as the installation of each agent is detected.

Apply default settings to define how the agents are protected. The settings include a protection scheme and strategy for the agents.

To enable the auto-creation of agents and define default settings:

Open the .

Go to Agents.

Click the button above the list of agents.

On the Auto declaration tab, enable the Automatically create agents option.

The newly discovered agents are automatically added to the list of declared agents. For more details on how to automatically assign new agents to tenants, see Assign New Agents to Tenants Automatically.
