About Server Replication

Server replication is a process which enables to secure both backed up data blocks and meta-data by replicating a Hyperstream server to another one.

If Server A fails, the backups of its agents are not redirected to Server B. However, as the replication of Server A to Server B has secured Server A blocks and meta-data, you can rebuild Server A from Server B. You can also restore the agents directly from Server B using the advanced restore function.
