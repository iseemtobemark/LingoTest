Perform a Combined Advanced Restore

To perform a combined advanced restore

Right-click the icon in the task bar and select .

Select Settings > Advanced.

Enable all three options. Provide the following information:

Remote agent Provide the following information to access the remote agent:

Fully Qualified Host Name.

Username.

Password.

Change server Enter the Fully Qualified Host Name of the server you want to restore from.

Perform a cross-restore Type the code of the cross-restore token received from your administrator (see Restore Tokens for details).

Select Connect.

The restore source and destination information is displayed under Connection status located in the upper right corner of the .

You can perform a folder or file restore in the same way as if restoring from your local machine.
