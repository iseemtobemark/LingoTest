Architecture

The BMR architecture includes (see Figure):

WinPE Simplified operating system serving as an emergency start-up platform that can be used to boot on the new workstation.

BMR restore wizard Wizard allowing you to choose the original agent and image to restore from the server.

BMR disk layout editor Graphical interface displaying the original disk/partition configuration, and the target disk/partition. The interface lets you change the original configuration to adapt to the new workstation.

Embedded agent used to restore data from the server to the new workstation, using a cross-restore.
