Update the Linux agent

You can activate the automatic update mode on the agents or update agents manually. See the Update URL section to configure the automatic update of agents in the . See the administration guide to configure the automatic update of agents in the .

To manually update agents

Open a terminal and log in as root.

Go to the directory where you downloaded the agent package corresponding to your system and launch the installation.

On Debian-based distributions, run:

dpkg -i <packagename>setup.deb

On RPM-based distributions, run:

rpm -U <packagename>setup.rpm
