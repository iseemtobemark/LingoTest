Activate the Service

The service is automatically started after the installation. You can also manually start it in the :

Launch a web browser and enter this URL: http://<your_aln_server>:8181/

Enter the name and password of a user having rights to the server (by default admin and admin).

In the (see Figure), select Settings > Preferences > License.

License view

In the License interface, click the link besides the Refresh button to access the Configurator.

Click the Configurator tile.

In the tab, select Enable to activate the product.

Click Apply, then close the Configurator.

The status of the service displays Configuration successfully applied.
