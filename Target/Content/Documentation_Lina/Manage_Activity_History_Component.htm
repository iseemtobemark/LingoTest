Manage the Activity History Component

The Activity History displays history information on backup and restore, global operations as well as on maintenance and replication. Activity Monitoring for details.

The Replication area displays the replication activity on <server_name> over the period of time you defined (by default, one day).

The volume of incoming data represents data replicated from another server to <server_name>, while the volume of outgoing data represents data replicated from <server_name> to another server.

No replication volume is directly attributed to nor since their replication does not involve any data transfer with the server.

The number of incoming or outgoing replicated blocks represents the number of blocks replicated to or from <server_name> per second.

The number of incoming or outgoing replicated objects represents the number of files and folders replicated to or from <server_name> per second.

The charts depend on the agentthe catalog, the unit (time) and the period selected in the left menu. If the periods proposed by the application do not suit, you can configure a customized period.

To configure a customized period

Click Activity History.

Click Custom Period.

Enter the information for display start date in the Display the last field and select the unit in the list (e.g., day(s), month(s).

Select the scale of values in the With a step of drop down list.

Select the present or a date to configure until when you want to see the statistics.

Click Apply.

Click the chart key to hide/show the corresponding information.
