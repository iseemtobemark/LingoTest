Display Details on an Agent

In the , click Agents.

In the list of agents, click on the Agent to display more details on the agent.

The agent details display. The agent details consists of four parts:

Properties: Information that is set up by an administrator when an agent is created.

Information: General information on an agent, such as the agent Operating System or the volume backed up on the agent.

Local configuration: Information on backup and network configurations collected from the agent, such as the agent network or the maximum acceptable data loss.

Local events: Information on the agent activity.
