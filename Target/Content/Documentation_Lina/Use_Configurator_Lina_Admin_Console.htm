Go to Settings > Configurator.

In the , you can:

Database tab: Modify, delete or add the directory selected for the ADE database.

Storage tab:

Modify or delete the directory selected for the database that stores the deduplicated data blocks.

Move content to an additional directory.

Hyperstream tab:

Activate HyperStream and HVDS after installing the server.

Modify or delete the directory selected for stream directories. A destination directory is required when a directory is created.

Move content to an additional directory.

tab:

Modify an existing directory

Deduplicated data blocks can be stored in one or several directories. To select a directory:

Open the .

Click Settings > Configurator. 

The Configurator window opens.

Click on the:

Database tab to move the content of the ADE database.

Storage tab to modify the existing volume and path selected for the database that stores the deduplicated data blocks.

Hyperstream tab to modify the existing volume and path selected for stream directories and add a destination directory.

tab to modify the existing volume and path selected for the metabase. It is possible to add paths in order to increase the storage, but a minimum of 1 TB of free space is required.

Select a volume, then enter a path.

If a destination already exists, you can use the ellipsis button to modify the path.

Click Apply. Your changes are saved.

Add a directory

Click Settings > Configurator > Database. 

The Configurator window opens.

To add a directory, click the Add a directory button.

Select a volume, then enter a path.

Click Apply. Your changes are saved.

If you define several repositories, agent databases are copied successively into one or the other repository. The server writes the first database into the first repository, then the next database into the next repository, etc.

Delete a directory

To remove a repository, click the Add a directory button then click the cross on the right hand of the path.

Removed repositories that contained agent databases are kept in the repository unless you move the database using the ade_server command.
