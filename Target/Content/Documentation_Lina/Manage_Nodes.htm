Manage Nodes

This section explains how to access the nodes list and modify a node.

Open the Nodes List in the

To open the nodes list

Go to the dashboard and click Total number of nodes or Server > Nodes.

In the list of nodes, you can:

Open a specific node in a separate window of the . To open a node in a separate window, hover on the node name, then click the button above the list of nodes or on the right-hand side of the window.

Refresh, filter or export the list of nodes in CSV or HTML format.

Display or hide specific information for all nodes, such as the master server used for each tenant or version of the node.

Modify a Node in the

To modify a node

Select the name of the node. The properties of the node display. You can:

Modify the node name and the access string used for the node.

View the master server and port number selected for the node.

View a summary of the node information displayed in the list of nodes, such as the Operating System used for the node.

Click Save.
