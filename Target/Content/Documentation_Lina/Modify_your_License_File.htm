Modify your License File

Request a License File add-on from Atempo.

Open the Atempo-License Manager configuration tool:

Windows. Select Start > All Programs > Atempo-License Manager > License Configuration.

Linux. Launch config_license.sh located in the Atempo-License Manager home directory.

Check the Add On option and:

Enter the path of the License file add-on you received.

 

Click Browse to locate the add-on.

Click OK.
