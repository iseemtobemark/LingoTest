Add an Agent Destination Directory

You can add a destination directory for a storage database, a stream directory or a metabase used for each backed up agent. To add a destination directory:

Select the directory to edit.

In the Data movement section, enable one or both of the following options:

Set new destinations as default.

Stop moving data if the size exceeds a specific threshold. You can set a customized threshold.

Click Add a destination directory, then enter a destination path.

Click Apply.

Restart the server to apply pending operations. You may have to log into the after the server restarts.

Click Yes to continue.
