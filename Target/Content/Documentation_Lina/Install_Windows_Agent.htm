Install a Windows Agent

Install a Windows Agent in Graphical Mode

Launch the program setup.exe on the agent from the directory: ATLXXX.iso\Agent\Windows\amd64

Follow the on-screen instructions:

License Agreement Accept the License Agreement to install .

Destination Folder Choose a directory other than the default (C:\Program Files\Atempo\Atempo Live Navigator\). Click Change... and select a directory in the tree.

Database Folder Choose the path to install the database of the . To choose a directory other than the default (C:\ProgramData\HNDB), click Change... and select a directory in the tree.

Ensure that you have at least 10% of the volume of the data to protect in free disk space for the database.

Network fully qualified name of the server Enter a fully qualified name for the server where the protection policy of your is defined. See Agent to Server Access Configuration for more details on the server name.

Click Install to finish installing.

Install a Windows Agent on a Remote Desktop Services Server

To install agents on an RDS server:

Open the command line interface.

Run the command change user /install.

Install the . The installation must be done once per RDS server.

When the agent is installed, run the command change user /execute.
