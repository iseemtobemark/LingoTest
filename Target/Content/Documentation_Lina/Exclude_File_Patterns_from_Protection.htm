Exclude File Patterns from Protection

To exclude file patterns from protection

Right-click the icon in the task bar and select .

The list of protections is displayed.

In the section File types excluded from the backup, click +.

Enter the pattern for the files to exclude.



*.avi to exclude all files with this extension.
~* to exclude all files beginning with "~".
SOFTWARE.LOG to exclude these files from any protected folder.

(Optional) If you want to apply rules in the BMR that exclude certain types of files, enter a more detailed path, for example /*/*.avi. The Bare Metal Recovery (BMR) is a module that performs a workstation recovery while applying rules to the system.

Click OK to confirm the creation of your exclusion. The Check column shows whether the validates the syntax you entered. See See Your List of Protected Files and Folders for details.
