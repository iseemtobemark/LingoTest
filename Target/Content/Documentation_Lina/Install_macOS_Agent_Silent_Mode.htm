Install a macOS Agent in Silent Mode

The deployment is performed with the LiveNavigator.pkg file:

In the same folder where the LiveNavigator.pkg is located, create a file named AutoConfig.txt containing this line:

SERVERNAME = <server_connection_string>

Replace <server_connection_string> with your Server information. The server connection string is in the form:

<servername>:<encrypted port>:<unencrypted port>

See Agent to Server Access Configuration for details.

Launch this command:

installer -pkg LiveNavigator.pkg -target "/"

The installation takes place at the root of your disk. The installation location cannot be modified.
