Prerequisites to macOS Agents Installation

Before installing  macOS Agents, check the following prerequisites (see Table):

Prerequisites to macOS agent installation

Prerequisite

Description

Operating System

To find out the list of supported Operating Systems, see the Compatibility Guide.

Memory

As required by the operating system.

Database directory

The database requires approximately 1% of the volume of the protected data in disk space, but can use up to 10% if you have a large number of files.

User Rights

The user performing the installation must have administrator rights on the machine.

Product Download

Ensure that you have downloaded the binaries from the Atempo Client Portal: https://support.atempo.com.

Communication Port

If you want to connect to the Agent interface remotely, for instance to perform remote restores, port 27261 must be open on your firewall.

Language

adopts the language set at this operating system level. Available languages are English and French. Ensure that you set the proper language for the OS.
