View the Last Protection Date of the

The last protection date is the last date at which you are able to restore data on your agent without losing any information. It does not mean that any data was backed up at that date.

If no new data has been added since the last session at the time a new session is supposed to occur (according to the RPO parameter), the Last Protection Date is updated even though no new data has been protected.

To view the last protection date of the agent

Hold the pointer over the icon in the task bar.
