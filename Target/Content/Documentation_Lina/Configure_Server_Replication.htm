Configure the Server Replication

The Replications view shows the replication server associated with the current or server and enables users to add or delete a replication destination. It can be viewed only by an administrator.

Ensure that you installed a server on all the machines involved.

The replication of the source server starts as soon as the replication destination is created. Ensure that the time to launch the replication is appropriate prior to creating the replication destination.
