To apply identical settings to several agents

In the , select Agents.

In the list of agents, select to the agents to be moved. A pencil icon appears above the list of agents.

Click on the pencil icon to edit settings for several agents.

Click on type of settings to change:

Protection: Enable the Agent group option, then select a protection scheme.

Strategy: Enable the Agent group and Strategy options, then select a strategy.

Block status: Enable the Block status option, then select a block status.

Update policy: Enable the Update policy option, then select an update policy.

Tags: Enable the Tags option, then select a tag.

E-mail: Enable the E-mail option, then select a user email.

Click Save.
