Add a Server Replication Destination

To add a replication destination

The replication starts as soon as the replication destination is created. Thus ensure that the time to launch the replication is appropriate prior to defining the replication destination.

Step 1: Set the required parameters

Open the .

Select Server > Structure section > Replications.

In the list at the top of the view, select one of the following options:

All tenants to create a global replication destination.

A specific tenant to create a selective replication destination.

Click the button located at the top of the Replication destinations view.

Enter the following parameters:

Name Name of the new destination, as displayed in the .

Port port on the target server.

It is the same port as Web administration port. The default value is 8181.

Select the Status toggle to enable the replication destination.

Enter the replication destination server.

(Optional) Enable the Use a proxy toggle if you want to reach the target server through a proxy.

Proxy server Host name of the proxy server you want to use.

Port Port of the proxy server you want to connect to.

Proxy user User of the proxy server if required by your proxy settings.

Password User password of the proxy server if required by your proxy settings.

Click Create.

Step 2 (Optional): Associate strategies and pools

Associating strategies and pools is only possible with a selective replication destination. With a global replication destination, all strategies are applied and all pools (if any) are used.

You can associate existing strategies or pools with your selective replication destination when creating a replication destination or after its creation.

To create a replication destination:

On the left pane, click Use.

The strategies and pools you have already added are displayed.

Select the strategies or pools you want to associate with the replication destination.

Click Create.

To associate strategies or pools with an existing destination replication:

Go to the list of replication destinations, then click your replication destination.

Repeat steps 1 and 2.

Click Save.
