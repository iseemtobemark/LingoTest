Protection options

The following protection parameters are available:

Pause Protection lets you manually suspend or resume data protection on your . The Pause mode is temporary: the backup resumes after 4 hours.

Boost Mode lets your temporarily accelerate data protection. This mode should not be activated permanently because it draws heavily on agent resources. Boost mode can be useful if your Agent has not been connected to the server over a period of time.

Recovery Point Objective (RPO) defines the maximum admissible data loss if data is restored between two protection sessions. If you modify the protection configuration on the server or on the agent, protection restarts immediately, regardless of the RPO time frame.
