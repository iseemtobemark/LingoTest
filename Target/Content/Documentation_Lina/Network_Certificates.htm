Network Certificates

In Secure mode (HTTPS), agents connecting to the server verify the server identity using these certificates:

Server certificate.

The name of the certificate file is server.pem, it is located in the Ref directory of the server home directory. For instance:

E:\Program Files\Atempo\HyperStream\default\HSX\ref\server.pem

Agent certificate.

The name of the certificate file is cacert.pem, it is located in the Ref directory of the home directory. For instance:
C:\Program Files\Atempo\Atempo \5.3.20991\Ref\cacert.pem

If you modify one of the certificates, you have to modify the other one as well to ensure that they match.
