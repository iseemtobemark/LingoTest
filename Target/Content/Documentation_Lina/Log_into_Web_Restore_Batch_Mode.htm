Log into Web Restore in Batch Mode

allows you to perform a as an administrator using the batch mode. This can be useful to restore files from a group of agents within a tenant. To perform a in batch mode, you need a password.

To access a tenant, you need a dedicated password. Make sure the tenant is logged in to the when connecting to the application. See Connecting to Lina Web Restore as Administrator for more details.

In your browser, enter the following URL:

https://<your_server_name>:8181/webrestore/batch

 

http://<your_server_name>:8181/webrestore/batch if you are on a secure local network.

You can also use the by selecting Restore > Web Restore > Web Restore batch.

Create a restore token. For more details, see Restore Token.

Enter the restore token.
