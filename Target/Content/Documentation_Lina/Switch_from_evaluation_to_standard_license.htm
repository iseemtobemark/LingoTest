Switch from an evaluation to a standard license

In the , go to Settings > Preferences > License.

Under the license type used, enter the license server name of an existing .

Enter a license server port number only if you do not want to use a port within the default range of 27000 to 27010. If you do not specify a license server port number, the server automatically assigns a free port number within the default range to communicate with the license server.

Port 27000 is not available, so port 27001 is chosen instead. The server does not attempt to find the next available free port when the one specified is already used

Click Apply license configuration.
