Uninstall the Linux Agent

Open a terminal and log in as root.

Launch the uninstallation:

On Debian-based distributions, run:

dpkg -P livenavigator

The command dpkg -P livenavigator uninstalls the agent and deletes the database. To uninstall without deleting the database, use the command:
dpkg -r livenavigator

On RPM-based distributions, run:

rpm -e livenavigator
