Remote Restore Mechanism

Remote restore enables you to launch a restore from a machine other than the one the data was backed up on. This type of restore requires a login that allows you to access the remote machine and to perform the restore.

macOS and Linux users must activate the SSH connection to allow remote access from another computer. On macOS this can be done via System Preferences > Sharing. You can select Remote Login and choose the user you want to have remote access.
