Modify a File Category

These are the syntax for specifying file types in a protection scheme:

*.extensionas in, for example, *.avi, *.mp3 or *.wmv to take into account all files with extensions of type "avi", "mp3" or "wmv".

~* to take into account all files beginning with "~".

software.log  to take into account all software.log files.

To modify a file category

Select Backup >Protections > File categories.

Click the file category to modify.

Add or modify the file names or extensions.

Click Save to save your changes.
