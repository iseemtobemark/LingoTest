Redirect Verification Messages

You can monitor verification messages without using the , and redirect them to a text file.

To redirect messages to a text file on a Windows system

Open a Command Prompt from the Start menu.

Go to the bin folder of the ADE installation directory.

To write the current messages to a file, run the command:

ade_admin.exe -u <user> -p <password> -A req > <file>

ade_admin.exe -u admin -p admin -A req > C:\messages.txt

To redirect messages to a text file on a Linux system

Open a Terminal.

Browse to the bin folder of the ADE installation directory.

To write the current messages to the /home/user/mess.txt file, run the command:

ade_admin -u <user> -p <password> -A req > <file>

ade_admin -u admin -p admin -A req > /home/user/mess.txt
