Launch the Linux

After the installation, the service is automatically launched on the agent.

To manually stop or start the service, launch the following command:

/etc/init.d/hnagent {start|stop|restart|status}

To launch the agent restore through the graphical interface (UI), click the Systray icon.

After the agent installation or update, you need to log off/log on to display the Systray icon and have access to the restore functions.

To manually launch the agent restore, launch the HNWizard.jar file located in the installation directory.

Debian 10 and CentOS version 7

You can manually launch the graphical interface (UI) in the Command terminal.

In the agent repository /opt/Atempo/HN/bin/AgentUI, launch the following command:

./agent-ui --no-sandbox
