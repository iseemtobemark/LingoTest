Limit the Agent Browse to the User Home Directory

When a machine is shared by multiple users, the administrator can define the backup configuration in the Strategy Details to prevent users from browsing the file structure of other users when logging on to the agent.

To limit the agent browse to user home directory

Open the .

Select Backup > Strategies.

Select a tenant in the Current tenant list at the top of the page.

Go to Agent > Agents rights, then enable the Limit agent browse to user home directory option.
