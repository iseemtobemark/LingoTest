Finder Extension

The agent provides the overlay mechanism to get the status of your protected files. Overlays appear on the directories included in the backup selection.

To activate overlays and the contextual menu on your Mac, you need to select the Finder extension (see Figure). Apple and third-party Mac extensions are managed in the Extensions System Preferences.

To activate the Finder extension

On your Mac, select Apple menu > System Preferences and click Extensions.

In the left menu, click Finder Extensions.

Activate Live Navigator Finder Integration.

Finder extension activated

If necessary, you can force the installation of the plugin with the following command: /Library/Application Support/HN/base/bin/finderpluginLoad.sh. Then, force the reboot of the finder. If another extension is installed, it is recommended to disable or remove it to avoid conflict.

Restart the hnagent service:

launchctl unload /Library/LaunchDaemons/com.hnagent.plist

launchctl load /Library/LaunchDaemons/com.hnagent.plist
