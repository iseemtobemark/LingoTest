View the Detailed Protection Status of an Object

To view the protection status of an object

Right-click in the Windows File Explorer on the folder or file for which you want to check the protection.

Select Backup > Check Protection.

A window displays the path of the selected object, as well as its protection status. You can hide or show the backup status, using the option below Check protection.
