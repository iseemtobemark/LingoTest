Undelete an Agent

When you delete an agent in the , its name is prefixed by @del- and it is no longer backed up. If you want to cancel the deletion, you need to do this within a certain period of time (by default one hour) before the agent is actually deleted.

To undelete an agent

In the , select Agents.

Select the deleted agents in the agent list and click Undelete agents (besides the Delete button).
