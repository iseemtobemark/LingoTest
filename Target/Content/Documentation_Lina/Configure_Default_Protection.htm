Set a default protection for new agents on a specific OS

A default protection can be applied on Windows, Linux or macOS agents. By default, this protection is applied when you create an agent.

Go to the Agents list.

Click on the button above the list of agents to configure the global agents parameters. These parameters apply to new agents and existing agents.

Go to Incoming agents > Default protections.

Set a default protection for the OS of your choice. If you do not want to set a default protection for a specific OS, you can select No server rule or exclude new agents from the protection.
