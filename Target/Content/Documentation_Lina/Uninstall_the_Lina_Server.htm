Uninstall the Server

Windows

In the list of applications installed on your workstation, select , then click Uninstall. A confirmation message appears.

Confirm you want to uninstall the server. The setup wizard appears.

In the setup wizard, confirm you want to uninstall the server.

(Optional) Restart your system.

The server is uninstalled.

Linux

Launch the script Uninstall_Product in the directory ADE_Home\Uninstall.
