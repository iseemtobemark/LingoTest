Perform a Remote Restore

To perform a remote restore

Right-click the icon in the task bar and select .

Select Settings > Advanced.

Enable Remote agent.

Provide the following information to access the remote agent:

Fully Qualified Host Name.

User name.

Password.

Select Connect.

The restore source and destination information is displayed under Connection status located in the upper right corner of the .

You can perform a folder or file restore in the same way as if restoring from your local machine.

Depending on the object to restore, see the following sections:

Restore Files.

Restore Folders.

By default, the remote connection is allowed, however the tunable HN_AGENT_ALLOW_ONLY_LOCAL = true can be added to the advsettings.cfg file (which must be in the HNDB directory) to prevent remote connection of machines.
