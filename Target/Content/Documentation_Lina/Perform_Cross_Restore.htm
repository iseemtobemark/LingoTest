Perform a Cross-Restore

To perform a cross-restore

Double-click the icon in the task bar.

Select Settings > Advanced.

Enable Perform a cross-restore.

Click the drop-down list to select the machine where the data to be restored is located.

The token window opens.

Type the code of the cross-restore token received from your administrator (see Restore Tokens for details).

Click OK to close the token window.

Select a machine. You can search a machine or browse the list of machines sorted in an alphabetical order.

Click Connect.

The restore source and destination information is displayed under Connection status located in the upper right corner of the .

You can perform a folder or file restore in the same way as if restoring from your local machine.
