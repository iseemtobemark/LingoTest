Upload the Advanced Environment Report to the Support FTP Server

An Advanced Environment Report (AER) can be sent to the Atempo Support team. This file contains information used to identify the cause of a reported issue, such as log or configuration files. To upload the file:

Rename the AER file and add the incident number assigned to you.

Use an anonymous connection to upload the file. You can use any third-party FTP client that supports passive mode (e.g., FileZilla).

Notify the Atempo Support team of the file upload.

For security reasons, the content of the FTP folder /pub/share cannot be listed.

Upload an AER File on Windows

Windows File Explorer

Specify the incident number at the beginning of the AER file name (e.g., 000654789_AER_servername-product_2021Oct10_10h10m10.zip).

Open Windows File Explorer.

In the path field, enter ftp://ftp.atempo.com/pub/share and press Enter.

Drag and drop the file in the window. If a file with the same name already exists on the FTP server, an error occurs.

Notify the Atempo Support team of the file upload.

Windows Command Line

Specify the incident number at the beginning of the AER file name (e.g., 000654789_AER_servername-product_2021Oct10_10h10m10.zip).

Open a DOS command prompt and go to the directory that contains the AER file.

Enter ftp ftp.atempo.com to open a FTP connection.

Enter anonymous as user name.

Enter your complete e-mail address as password.

Enter cd /pub/share to change directory.

Enter bin to set the transfer mode as binary.

Enter mput your_AER_file.zip to upload the AER file.

Enter quit to close the connection.

Notify the Atempo Support team of the file upload.

Upload an AER File on Linux

Specify the incident number at the beginning of the AER file name (e.g., 000654789_AER_servername-product_2021Oct10_10h10m10.zip).

Open a terminal.

Enter the directory name that contains the AER file.

Run the following commands:

open ftp.atempo.com

user anonymous support

cd /pub/share

bin

put your_AER_file.zip

quit

EOF

Where: your_AER_file is the name of the AER file.

Press Enter. The file upload may take some time.

Notify the Atempo Support team of the file upload.
