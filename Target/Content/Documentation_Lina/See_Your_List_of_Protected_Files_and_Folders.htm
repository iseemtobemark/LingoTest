To see your list of protected files and folders

Right-click the icon in the task bar and select . The list of protections is displayed.

The interface is divided into two blocks (see Figure):

The left block contains the server rules defined by the administrator.

The right block presents My backup settings. It contains:

Items included in the backup.

Items excluded from the backup.

Files excluded from the backup.

If the server has granted permissions, the user can configure the file and folder protections on the agent

In the Check column of your backup settings, the displays if the protection is compatible with the rules defined on the server (see Table).
