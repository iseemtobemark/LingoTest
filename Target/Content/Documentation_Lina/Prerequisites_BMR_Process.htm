Prerequisites to the BMR Use

The BMR module can be used on bootable media such as CDs, DVDs, or USB flash drives. The bootable file delivered with the product should be kept in a convenient location and readily available in the event of a BMR operation.

The firmware between the machine you backed up and the machine you are restoring to must be identical (i.e., BIOS, EFI, UEFI).

Before starting the BMR process, you must:

Check the Protect the system with BMR option in the protection scheme.

Obtain a token for each agent to recreate. For more information, see Restore Tokens.

If Bitlocker encryption is supported you must use new disks or you have to format the existing disks before proceeding with the restore.
