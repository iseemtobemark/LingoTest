Create and Configure a User Group

A user group is a set of users associated with a user profile. Depending on their profile, they will be able to access or not to the functions of the application.

In the list of user groups, you can:

Delete or duplicate a user group.

Refresh the list of user groups.

Export one or several user groups in CSV or HTML formats. By default, the list of all user groups is exported.

To create a user group

Open the with administrator rights.

Select Server > Users > Users.

Click on the User groups button besides the filter bar.

The list of user groups displays.

Select a tenant to which the user should be assigned.
