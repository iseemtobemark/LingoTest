Install a Windows Agent in Silent Mode

The silent mode lets you install, upgrade, or uninstall from the command line, without having to launch a graphical interface.

Launch a command line on your agent and navigate to the directory containing the setup.exe program of your : ATLXXX.iso\Agent\Windows\amd64

Enter this command:

setup.exe /q /CompleteCommandArgs "SERVERNAME=xxxx DATABASEDIR=xxxx INSTALLDIR=xxxx"

If the value contains space characters, enclose it between escape quotes (\").

For instance:

setup.exe /q /CompleteCommandArgs "SERVERNAME=lina-server DATABASEDIR=\"C:\Program Files\ALN_Agent\""

The installation in silent mode consists of multiple parameters (see Table).

Installation parameters in silent mode

Parameter

Description

Status

INSTALLDIR

installation path.

Optional, the default value is C:\Program Files\Atempo\Atempo Live Navigator\

DATABASEDIR

database path.

Optional, the default value is C:\ProgramData\HNDB

SERVERNAME

Connection string of the Server where the agent data is going to be protected.

The server connection string is in the form:

<servername>:<port>

To indicate the tenant:

<tenant_name>/@<servername>

See Agent to Server Access Configuration for details.

Mandatory

- /l c:\logfile.log

Creates a log file in c:\logfile.log.

Optional

- /qb

The setup only displays a small window with progression information.

Optional

- /q

The setup is hidden.
