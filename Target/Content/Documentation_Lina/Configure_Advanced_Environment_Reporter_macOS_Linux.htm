Configure the Advanced Environment Reporter on macOS or Linux

Under macOS and Unix, advanced_env_report does not have a graphic interface. When you launch the shell, all Atempo software environments present on your system are detected and presented as numbered options. These are all the Atempo software environments that the shell can detect (see Table):

[root@tonga Linux-X64]# /opt/Atempo/HN/bin/advanced_env_report.sh

2) _Aws (_AWS)

Selecting any option toggles between the default and the alternative (e.g., selecting option 1 sets it to Yes, whereas selecting option 2 sets it to No).

If you want to exit the without running it, press CTRL+C on the command line.
