Assign Rights to LDAP Groups in Command Line Mode

On the machine hosting the , launch the ade_admin command line interface:

Windows Select Start > Programs > > Administration.

macOS/Linux Launch the command ./ade_admin.

Enter the password of the admin user.

Launch the command user add_ldap_group “<group_name>”.

user add_ldap_group “adeadmin2. ALN.Groups.16 - R&D ALN@AD.fr”
