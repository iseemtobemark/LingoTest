View your Protected Volumes

To view your protected volumes

Right-click the icon in the task bar and select .

Select Settings > Volumes.

To view the volumes that are supported by , check Show only supported volumes. You can use the Refresh button to refresh the list of supported volumes

Optional Volumes contains the list of the offline volumes that are currently connected or have been protected at some point in the past.

Link volumes contains

The following parameters on current protected volumes are displayed (see Table).

Protected volume parameters and their description

Parameter

Description

Protected or Activated

Shows a green or grey check mark to indicate that the volume is currently protected by .

Mount Point

Indicates letter of the volume on Windows.

Name

Shows the volume name.

Supported

Shows a green check mark if the volume is supported.

Id

Allows to recognize a volume in order to resume protection.

Path

Displays when multiple mount points are detected for a volume. It indicates the mount points.

Reference Path

Displays when multiple mount points are detected for a volume. The reference path indicates the actual path taken into account to apply the protection scheme parameters and to decide what to protect on the volume.
