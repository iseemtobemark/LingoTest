Assign New Agents to Tenants Automatically

To automatically assign new agents to a tenant:

Open the .

Go to the Agents list.

Click the button above the list of agents. then click Incoming agents.

In the Auto assignment tab, click Add an auto-assignment option.

Define the auto-assignment parameters:

Select the parameter used to assign the agent to a tenant.

Agents are assigned to a tenant depending on the agent name, the IP address of the agent, its system category or Windows domain.

Specify a string value for the parameter.

Specify the tenant that will contain the agent. This field may not appear if you do not have several tenants.

Specify the agent group that will contain the agent or select No protection inheritance.

Specify the protection and strategy to apply to the agent.

Click on Save.

The order in which the parameters are entered is important; it corresponds to the order of priority in which they are applied.
