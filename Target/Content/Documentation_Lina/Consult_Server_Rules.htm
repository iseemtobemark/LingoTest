Understand the Server Rules

About Server Rules

The diagram (see Figure) contains two paths:

C:\Doc Files

C:\Templates

Each path submits to a default action that may have exceptions. This default action is symbolized by an icon at the root of the path.

C:\Doc Files: Include, user cannot make changes.

C:\Templates: Exclude, user cannot make changes.

Both paths contain file categories that are either included in or excluded from protection:

C:\Doc Files:*.avi, *.docx, *.mp3 Exclude, user cannot make changes.

C:\Templates:*.csv, *.doc, *.docx Include, user cannot make changes.
