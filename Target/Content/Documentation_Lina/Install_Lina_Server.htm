Install the Server

Launch the setup program corresponding to your operating system:

Windows Double-click the executable ATLXXX-Windows-X64.exe.

Linux Run ATLXXX_Linux-X64.bin. You need to ensure that the binary can be executed. If necessary, you can define the rights with the command: chmod +x

Select the installation language, then click OK.

Click Next.

Accept the license agreement, then click Next. The  Server is included by default in the setup program.

You can install the at a later date. To do so, relaunch the present setup, choose Installation then select the in the product selection window.
Whether you install Atempo-License Manager or not, the Server starts with a 30-day evaluation license. See Install the [%=ALN.license_doc%] for details on how to switch to a permanent license.

The main destination directory for the setup. The default destination directory is C:\Program Files\Atempo. To return to the default directory, click Restore Default. It is recommended to install on a physical machine rather than a virtual one.

Click Install.

When the installation is finished, the Atempo Deduplication Engine service starts up. For more details on a server configuration, see Use the Configurator in Wizard Mode.
