Reports

A report contains data on agents. It can be used by an administrator to follow the activity of all of the agents declared for a tenant. To follow the activity of an agent, see Presenting [%=ALN.aln%] Agents. A report can be accessed through the Administration console or can be sent by e-mail. This section explains how to:

View a report.

Download a report.

Send a report.

Delete a report.

View a Report

Users may view a part of the reports, depending on their access rights. A superadmin can consult the reports on all of the agents, whereas administrators can consult only the reports of the agents of their tenant. To display the reports associated with one or several report templates:

In the Server tab, click Statistics & Reporting, then Backup reports.

In the list of report templates, select one or several report templates, then click to generate reports.

In the list of report templates, click to display the list of generated reports.. It is possible to filter reports, using the name of the report or report template, or the generation date of the report.

Download a Report

In the Server tab, click Statistics & Reporting, then Backup reports.

In the list of report templates, click to display the list of generated reports.

In the list of reports, select a report, then click . The report is generated as an HTML file. To select all the reports, click the checkbox above the list of reports.

Delete a Report

In the Server tab, click Statistics & Reporting, then Backup reports.

In the list of report templates, click to display the list of generated reports.

In the list of reports, select a report, then click . To select all the reports, click the checkbox above the list of reports.

Confirm the deletion of the report. The list of reports is automatically updated.

Send a Report

Reports can be sent automatically or manually. To send a report automatically, see Report _Ttemplates. To send a report manually:

In the Server tab, click Statistics & Reporting, then Backup reports.

In the list of report templates, select a report template, then click .

To add an e-mail address in the list of recipients:

Open the report template, then go to Reports > Sending.

Enter an e-mail address, then clickAdd.
