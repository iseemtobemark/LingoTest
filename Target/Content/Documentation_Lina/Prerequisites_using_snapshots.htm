Prerequisites to using snapshots

Windows:

All volumes to snapshot must be stored on NTFS file systems.

If you want to use the snapshot revert feature, do not install any ADE components on the system disk.

Administrators must be familiar with Windows snapshot procedures.

Linux:

All volumes must be stored on version 2.02.95 or higher of LVM2.

All volumes used by ADE must be Thin Provisioned LVM volumes (thin volumes). See Creation of a Thin Volume on Linux.

Administrators must be familiar with Linux snapshot procedures.
