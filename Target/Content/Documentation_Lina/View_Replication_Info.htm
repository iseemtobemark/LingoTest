View Replication Information

You can display replication information in the Agent view of the and in command mode.

To display replication information in the

In the , select Server > Structure > Replications. You can see standard information on replication destinations:

Name of the replication destination.

Status of the replication destination.

Name of the server used for the replication destination.

Port number used for the replication destination.

Click the button to display additional information in the list of replication destinations. Check the replication columns you want to display. You can see information on whether a replication destination is used globally and if a tenant is shared.

To display replication information in command line

On the machine hosting the :

Windows.

Select Start  Programs  Atempo-Deduplication Engine  Administration.

Linux.

Launch the ./ade_admin command.

Provide a password. Two default users exist in the :

admin, password admin (default user with administrative rights)

tina, password tina (standard user)

If you must launch the Command Line Interface with a different user, then run the ./ade_admin -u username command.

You are prompted to enter the password for this user.

Run the repli_info command to display this information:

The replication destination.

The replication status.

The replication lag can be inferred from the last synchro point which is the date from which you are able to restore data from the replication target if the replication source is off. It is the difference between the current time and the last synchro point.
