Stop the Deduplication Engine service (by default ADE.default).

Launch this command: ade_server --base_path=<path>

Where <path> is the new database and index path.

Launch this command: ade_server --log_path=<path>

Where <path> is the new database log path, typically, a subdirectory of the database directory set in step 2.

All database files are moved to the new path.

The database should be located on a local SSD disk.
