Restore Data on Synology

Data restore on Synology can be performed with:

A cross or remote restore via the installed on another machine.

A web restore via a browser.

For more information, see Cross-Restore, Remote Restore or Web Restore.

For more information, see Cross-Restore Mechanism, Remote Restore Mechanism or Web Restore.

To perform a cross or remote restore, the version must be equal to the one installed on the Synology Agent. The server version can be equal to or higher than the version.
