Report Templates

A report template is used to configure the content of a report. This chapter explains how to:

Create a report template.

Edit a report template.

Delete a report template.

In the list of report templates, you can create a report template or select a report template, then:

Delete or duplicate a report template.

Generate reports for a specific report template.

Send reports generated for a specific report template.

Display reports generated for a specific report template.

Create a Report Template

Step 1: Define a name for the report template

In the Server tab, go to Statistics & Reporting > Backup reports.

In the list of report templates, click .

Define a name for the report template. By default, the name is New report template. The name of a report template can be defined during or after creation.

Step 2: Configure the report template

A report template can be configured during or after creation, using two tabs:

The Report templates tab: This tab is used to define the content of the report.

The Reports tab: This tab is used to schedule automatic reports.

Edit the Reports tab (see Table).

Parameters in the Reports tab

Option

Description

Enable automatic e-mail report

 Defines whether the report should be sent automatically.

<Schedule frequency> on <day of the week> at <server time>

Schedules the recurrence of the report. A report can be sent every day, week or month. A weekly report is sent on every Monday, whereas a monthly report is sent on the first day of the month.

Send mode

Defines how the report should be sent. A report can be:

Enclosed as a file attachment.

Embedded as a URL link.

Embedded as a static table.

Enter an e-mail address

Defines a list of recipients for the report. Each recipient should be separated by a comma. The SMTP Mail Service must been configured, otherwise the content of this tab will not be editable. For more details on the configuration of the SMTP Mail Service, see Configuring the SMTP Mail Service.

Step 3: Confirm the creation of the report template

Click Create. A pop-up dialog confirms the report template is created.

Delete a Report Template

To delete a report template, as well as the reports associated with that report template:

In the Server tab, go to click Statistics & Reporting > Backup reports.

In the list of report templates, select a report template, then click .

Confirm the deletion of the report template. The list of report templates is automatically updated.
