Uninstall a Windows Agent

Ensure that there are no other users of currently connected to the machine.

From the Windows Start menu, select Control Panel > Programs and Features > Uninstall or Change a Program.

In the list of programs, double-click Atempo  and click Next. A pop-up window asks you to confirm the uninstallation.

Select Remove then Next.

Confirm the deletion of the Atempo database.

Restart your host to complete the uninstallation.
