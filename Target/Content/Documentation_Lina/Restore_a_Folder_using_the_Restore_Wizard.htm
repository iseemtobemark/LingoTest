Restore a Folder Using the Restore Wizard

Step 1: Launch the

Right-click the icon in the task bar and select .

Select Restore a folder > Start.

Step 2: Locate the folder to restore

In the folder selection window, select Browse backup to locate the folder you want to restore. You can activate Search in subfolders to broaden the search. You can indicate either the complete path, or enter a parent folder and choose to restore subfolders in the tree.

Click OK to close the Backup window. The folder path appears in the folder selection window.

Click Next.

Step 3 (optional): Specify time and date ranges

If you know the last modification of your folder, modify the time and date ranges.

Select Next.

Step 4: Define restore options and restore your folder

Choose the restore location.

By default, the folder is restored in the same location, but you can choose a new location:

Turn on the toggle to change the target directory of the restore.

Click on Browse computer to add the target directory, then click Select Folder.

Choose whether to restore the folder and its content or only the content.

If the folder to restore already exists at this location, select the restore behavior:

Overwrite The folder is replaced by the restored folder.

Skip The folder is not restored.

Rename The folder already present is renamed with the suffix .bak.

If needed, change the identity to a user that may have more privilege to restore data and click Change identity. If the chosen identity is linked to an admin group, the restore is done directly as administrator (NT Authority\system on Windows, root on Linux and macOS):

Impersonated restore Type the username and password of the user you want to restore.

System restore on Windows Type the username and password belonging to the local administrators group.

System restore Linux and macOS Type the username and password belonging to the admin, adm, sudo, wheel or root groups.

Click Check Identity.

Select Restore. To cancel the restore process, click Cancel.

Once the restore process is complete, you can view the details in the status overview.

If the folder contains more than 10,000 files, for performance reasons, the list of restored files is not displayed in real time.

Click Finish to close the . You are redirected to the home page of the .
