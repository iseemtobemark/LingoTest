Restore Data from the Agent Interface

The agent interface allows you to restore any protected version of a file or folder backed up on the server. This can be useful when an object has been corrupted, lost or accidentally overwritten.

To perform a restore, you need to make sure that the agent onto which you want to restore data is connected to the server through a network.

In addition to restoring your data on your own local machine, enables you to perform the following types of advanced restore:

Cross-Restore: Restore data to a computer other than the one where data was backed up.

Remote Restore: Launch the restore from a computer other than the one where data was backed up.

Restore from an Alternate Server: Restore data backed up by a server other than the one backing up your agent.

Combined Advanced Restore: Perform a cross-restore on a remote machine of data stored on a replicated server.
