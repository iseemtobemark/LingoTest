Create a Protection zone

In the list of protection zones, you can:

Display the list of protection zones per name, tenant, path and use.

Create, edit, delete and duplicate protections.

Refresh the list of protection zones.

Copy protection zones to another tenant. To copy a protection zone to another tenant:

Select the protection zone, then click the button shaped like a globe.

Select the tenant that will receive the protection, then click Save.

Export a list of protection zone in .CSV or .HTML formats.

To create a protection zone

Open the .

Select Backup > Protections zones.

Select a tenant for the protection zone.

Click .

In the General section, enter a name for the protection zone to declare.

In the Path section, enter the syntax for the system path on which you want to apply a protection rule.

Click Create to complete the creation of your protection zone. It is then displayed in the list of protection zones.
