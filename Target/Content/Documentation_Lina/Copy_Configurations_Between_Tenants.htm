Copy configurations between tenants

To facilitate administration, you can copy an entire configuration of a tenant to one or more other tenants. This concerns the following elements:

Strategies.

Protection policies, such as rules, paths, and file categories.

User profiles.

In the , if you want to copy:

Strategies, files categories, protection policies, rules or zones, go to: Backup > Protection configuration, then select the type of element to copy.

User profiles, go to Server > Users > User profiles.

Select the element you want to copy.

Click Copy the selected <type of element> to other tenants (globe shaped-button).

Check the tenant(s) to which you want to copy.

Click Save.
