Create an Agent Group

An agent group is used to group agents that are managed by the same administrator.

Open the .

Go to Agents, then click on the Agent groups button above the list of Agents.

Select a tenant.

Click .

Enter the name of the agent group.

Select a protection scheme.

Select a strategy.

Click Create to complete the creation of the group.

The group displays above the default group (No protection inheritance).

To edit an agent group

Click on the agent group in the list of agent groups.

To assign an agent group to agents

Go to the agent details if you want to edit the properties of an agent: For more information, see Create [%=General.ProductName%] Agents Manually.

Go to the agents list if you want to edit the properties of several agents:

Select one or several agents, then click the pencil icon.

In the agents properties, an option called Agent group appears. By default, this option is disabled.

Enable the Agent group option, then select an agent group.

By default, agents inherit the protection and strategy set for any agent group other than the default agent group.

If the default agent group is selected (No protection inheritance), you can select specific protections and strategies for the agents. In that case:

Enable the Protection and Strategy options.

Select a protection rule and a strategy for the agents.
