Change the Validity Duration of the Password

By default, the security code sent to you is valid for 1 hour. To modify the duration, you need administrator rights.

Open the .

Select Server > Preferences.

Select the Tunables tab.

Activate Show advanced parameters. The list of tunables is displayed (see Figure).

Tunables list showing the advanced parameters

Double click the variable name webrestore.max_password_age.

The field becomes editable.

Enter the validity duration in minutes. You can do the following:

If you enter 0, the security code will never expire.

The minimum validity duration is 5.

Any number between 1 and 5 will be recorded as 5.

Click outside of the field to validate the new value.
