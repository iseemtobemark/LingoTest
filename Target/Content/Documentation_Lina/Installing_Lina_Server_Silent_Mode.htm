Install the Server in Silent Mode

If you cannot install in graphical mode, you can install/upgrade in silent mode.

To do so, you must create a batch file and use this file to install on machines without graphical or X terminals.

To install in silent mode

Open the batch_install_sample.txt sample file located in the Server directory of the product iso.

Set the value of the required parameters according to your environment and save the batch file as a text file.

To edit the file in Windows, use a text editor such as Wordpad or Notepad++, not Microsoft Word or equivalent as it may corrupt the file syntax.

Use double backslashes in Windows paths.

The sample below is set to install .

Launch the installer using this command:

<installer_path> -f <batch_file_absolute_path>

For instance:

Windows.

C:\temp\ATLXXX-Windows-X64.exe -f C:\temp\my_batch_file.txt

Linux.

/tmp/ATLXXX-Linux-X64.bin -f /tmp/my_batch_file.txt

install starts automatically with the parameters specified in your batch file.
