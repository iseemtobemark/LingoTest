Session Timeout

When you are logged to the , the Web Restore or ADE Statistics interfaces, your session will expire once the session timeout is reached. By default, the timeout is set to 4 hours (14 400 seconds). When you have reached the timeout, you need to login again to the web interface.

To change the value of the timeout

Open the .

Select Settings > Tunables > Advanced > Web server.

In httpd.session_timeout, enter the validity duration in seconds.

The new value is automatically saved.
