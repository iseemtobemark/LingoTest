Install the in Synology DSM 6

To install the Synology :

In the Package Center, click Installed in the left menu pane.

In Settings under the General tab, check Any publisher to enable the installation of packages published by any publisher.

Check your Synology information in the Package Center (see Figure).

Basic Synology information

Click Manual Install.

Select the package corresponding to your Synology model. If you do not select the appropriate package, an error message displays.

Enter the work directory in which the agent will be installed.

Enter the server name.

Click Next.

Click Apply to confirm the settings. The installation is complete (see Figure).
