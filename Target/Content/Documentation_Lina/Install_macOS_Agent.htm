Install a macOS Agent

Go to the product download directory ATLXXX.iso/Agent/MacOSX.

Double-click the LiveNavigator.dmg file, then the LiveNavigator.pkg file.

Enter the following parameters:

License Agreement Accept the License Agreement to install .

Server Name Enter the name of the machine hosting the server.

Database Folder Lets you choose the path to install the database of the . Ensure that you have at least 10% of the volume of the data to protect in free disk space for the database. To choose a folder other than the default (/Library/Application Support/HN/DB), click Browse and select a folder in the tree.

Login Information Provide the name and password of the administrator.
