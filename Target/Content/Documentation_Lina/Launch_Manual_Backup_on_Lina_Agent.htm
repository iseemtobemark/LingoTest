Launch a Manual Backup on the

You can launch a manual backup on your , using the File Explorer. A manual backup can be launched:

On a file or a folder saved on your workstation.

As a session separate from a backup session scheduled by an administrator, which means you can perform a manual backup at any time. The manual backup is added to the list of backups performed on your agent.

To launch a manual backup on a

In the File Explorer, right-click on the folder or file that you want to back up.

If your OS is:

Windows and Linux: Select Backup > Backup now.

MacOS: In the menu, select Backup now.

A window containing the backup status as well as the location of the item backed up displays.

Click OK or close the window.

For more details on how to restore a file or a folder, see Restore Files in Browse Mode or Restore Folders.
