Configure Agent Protections

A number of protection schemes are predefined in the software to help you configure data protection. You can nevertheless create new ones as needed. This section explains how customize data protection.

As described in Defining Data to Protect, to create a protection scheme, you need these items:

Protection selections Associate a path and a protection rule.

Paths Define objects to protect.

Protection rules Indicate the types of files to protect.

File categories Set file categories on which to apply the protection rules.

Actions Specify to include or exclude files from protection and to determine to what level users can make their own choices on their agents.

To customize the protection of your agents, begin by creating paths and categories. Then, you can create new protection rules and selections and finally a protection scheme.
