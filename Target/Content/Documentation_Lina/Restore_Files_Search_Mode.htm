Restore files in Search Mode

Connect to . See Accessing [%=ALN.aln_web_resto%].

Click the Search button in the upper left corner. The search view is displayed (see Figure).

Search view parameters

Enter you search criteria:

Modification Date: Select the time interval within which the file was modified by using the drop-down menu or the Advanced view.

Names and Extensions: Type your own file name or extension and then click Add or click Select to choose one of the predefined file categories.

Path: Type in the path where you want to search or click Browse to select the path.

To modify or remove the path, click Clear next to the path field.

Once you have entered your search criteria, select Search. The corresponding results are displayed.

If the search returns too many files, narrow-down your search criteria.

Click Show File Versions.

In the Versions window, select the version (see Figure).

The versions icon is located in the column left of the file name

Select the icon to start the restore.

Choose whether you want to open/view or save/save as the restored file.

Click Close in the Restore Complete window.
