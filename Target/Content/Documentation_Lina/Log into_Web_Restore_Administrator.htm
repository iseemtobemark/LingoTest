Log into Web Restore as an Administrator

Depending on your browser settings and on whether or not you have created your own SSL certificate, you may get a warning message concerning the security of the web page you are accessing. See SSL Certificate for details.

In your browser, enter this URL:

https://<your_server_name>:8181/webrestore/admin

 

http://<your_server_name>:8181/webrestore/admin if you are on a secure local network.

To access Web Restore in the , select Restore > Web restore > Web restore

Create a restore token. For more details, see Restore Token.

Enter the restore token.

Select Login to access the application.
