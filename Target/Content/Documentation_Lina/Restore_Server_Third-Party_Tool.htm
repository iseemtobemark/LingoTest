Restore the Server with a Third-Party Tool

To restore the Deduplication Engine database backed up with a third party tool

Stop the ADE service.

Restore the database with the third party tool.

Create a file named .tina_restored at the root of the Deduplication Engine installation directory. This file can be empty. It is automatically removed by ADE once the restore process is complete.

Restart the ADE service.

Regardless of the method you choose, after a restore of your server, you need to manually remove the file db_backup_sync located in the database path.
