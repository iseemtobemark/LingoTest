Activate the Protection Status of Files and Folders in the File Explorer

On Windows agents, you can use the overlay feature to display the protection status of files and folders in the File Explorer.

Right-click in the Windows File Explorer on any folder or file and select Atempo  > Show backup Status.

A green icon indicates that the folder of file is protected. A red icon indicates that the folder or file is not protected.

Refresh the File Explorer to see the icons.

To disable the overlay, right-click any folder or file and select Hide backup status. uses six different overlay icons. By design, Windows only supports ten overlay icons. If other applications are already using overlay icons, you may not be able to see icons. Check the overlay used in the registry key: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\ CurrentVersion\Explorer\ShellIconOverlayIdentifiers. If needed, delete other application overlay icons.
