Server Configuration View

The Server Configuration view shows information related to the server, i.e., Health Check, Forecast, and Server Information.

To display the Server Configuration view

Click Server System Information System Status.


The Server tab contains these elements:

Forecast area. Shows this information:

The disk space available on the server.

The percent of disk space used.

The growth in disk space used over the last week.

An estimated number of days when the disk is full. The estimate is based on last week growth and the available disk space.

 Server Information. Shows this information:

Information on restorable and stored volume.

Deduplication Ratio: Current deduplication ratio for the server.

CPU: Instant CPU usage or CPU global load for the server system.

RAM: Current RAM allocated to activities.
