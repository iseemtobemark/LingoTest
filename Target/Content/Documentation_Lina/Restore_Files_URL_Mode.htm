(Optional) If you want to restore files at a certain point in the past:

Use the time slider to choose the date at which you want to restore your files.

 

Use the calendar view to pick the restore time.

Once you have chosen a time in the past, displays your agent file tree at that date. The maximum time interval corresponds to the retention defined for the agent. This is to ensure that you have a coherent view of your agent in the past.

Click Generate URL.

Copy the URL (s) you want to send for agent restore(s).

Paste the URL(s) in the e-mail.

Send it to the recipient. The URL can be protected and four of its digits can be replaced by "XXXX". The missing information is sent to the recipient by SMS message or any other way.
