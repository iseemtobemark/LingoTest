Configuration

The enables you to set the configuration parameters that only apply to . These parameters are set in the list of Agents.

To access the list of agents

Go to the Agents list.

Click on the button above the list of agents to configure the global agents parameters. These parameters apply to new agents and existing agents.

Set a default strategy for new agents

Go to Default strategy.

Set a default strategy for the OS of your choice.

The standard strategy is Default strategy / No restrictions / With network encryption.

Create a tag for declared agents

A tag can be used to:

Sort agents that share a common characteristic (a location, for instance). Tags can be added in the agent details or in properties applied to all agents.

Filter agents in the list of agents.

Search several agents without using the filter in the list. No history selection is retained when using the filter, which means the agent you select is retained until the filter is used to search another agent.

To create a tag

Click in the tag field and enter a tag name.

Pick a tag color, then click the + button.

The tag is automatically added to the list of tags used in the .

To edit the name of an existing tag, click on the tag, then edit the tag name. To delete an existing tag, click the bin button in the tag.

To select an existing tag for an agent

Go the agent details, then select one or several tags. For more details, see Create [%=General.ProductName%] Agents Manually.
